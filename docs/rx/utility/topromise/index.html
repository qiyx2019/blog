toPromise

函数签名: toPromise() : Promise​

将 observable 转换成 promise 。​

⚠️ toPromise has been deprecated! (RxJS 5.5+)

示例​

示例 1: 基础的 Promise​

( jsBin | jsFiddle )

// 返回基础的 observable
const sample = val => Rx.Observable.of(val).delay(5000);
// 将基础的 observable 转换成 promise
const example = sample('First Example')
  .toPromise()
  // 输出: 'First Example'
  .then(result => {
    console.log('From Promise:', result);
  });
示例 2: 使用 Promise.all​

( jsBin | jsFiddle )

// 返回基础的 observable
const sample = val => Rx.Observable.of(val).delay(5000);
/*
  将每个 observable 转换成 promise 并使用 Promise.all 
  来等待所有 promise 解析完成
  (你或许应该使用 forkJoin，而不是 toPromise)
*/
const example = () => {
  return Promise.all([
    sample('Promise 1').toPromise(),
    sample('Promise 2').toPromise()
  ]);
}
// 输出: ["Promise 1", "Promise 2"]
example().then(val => {
  console.log('Promise.all Result:', val);
});
